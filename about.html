About This To-Do App

What is this App?

The To-Do List Application is a smart, intuitive, and fully functional web-based task management tool designed to help users stay organized and productive. Built using HTML, CSS, JavaScript, PHP, and MySQL, this app allows users to easily create, manage, and track their daily tasks from a clean and responsive dashboard.
At its core, this application is more than just a simple task list. It is a thoughtfully crafted productivity tool that empowers users to structure their daily routines, prioritize important work, and gain clarity over their responsibilities ‚Äî whether it's for personal goals, academic work, or professional projects.

Key Features

User Registration and Login (with password hashing)

Add, Edit, Delete (Soft Delete) and Complete Tasks

Set Task Deadline and Priority (High, Medium, Low)

Search, Sort (A-Z, Z-A, Deadline Asc/Desc), and Filter Tasks by Priority

Responsive UI with Dark Mode toggle

Drag & Drop Task Reordering

Export Tasks as CSV, PDF, and JSON

User-specific task management with secure session authentication

Task Details view page

Toast notifications and alerts

Secure against common vulnerabilities (XSS, CSRF, SQL Injection)

Pagination support for tasks

Profile page with user info

Tech Stack

Frontend: HTML, CSS, JavaScript (AJAX)

Backend: PHP (Vanilla), MySQL

Other: FPDF (PDF Export), LocalStorage, Session Auth

How It Works

Users can:


Add tasks with titles, descriptions, and due dates


View all tasks as individual cards on a dashboard


Count words in each task description for clarity


Click on a task card to view full details on a separate view page


Track completed or pending tasks


Sort and organize tasks by date or priority (if implemented)


All the data is stored securely in a MySQL database, with dynamic interaction handled using PHP on the server-side and JavaScript on the client-side.


Data Privacy


All user tasks are stored securely in the database, with no third-party tracking or external API sharing. This ensures the app runs offline on localhost or your own server, giving full control to the user.
